| Authentication | Gong's auth is Access Key and Secret based with an organization URL. |
| External App | Not needed for Gong, API access is directly from access key and secret |
| Docs | [Gong docs](https://help.gong.io/docs/receive-access-to-the-api) |

<Note>
  If you're using Stoneturner's Managed MCP, you do not need to create an external app.

  If you're running Stoneturner locally, you will need to create an external app for OAuth apps to go through the authorization flow.
</Note>
